# Social Media Clone

## Overview
This project is a social media clone that replicates some of the key functionalities of popular social networking platforms. The aim is to provide a comprehensive learning experience in full-stack development, including front-end, back-end, and database management. Key features include user authentication, post creation, interaction capabilities, and a responsive design.

## Features
- **User Authentication and Profiles**: Secure user sign-up and login using JWT, with user profile management.
- **Post Creation and Interaction**: Create, edit, delete posts; like, comment, and save posts.
- **Public and Private Accounts**: Users can set their profiles to be public or private.
- **Followers and Following**: Users can follow each other to see posts in their feed.
- **Follow Request Management**: Accept or reject follow requests for private accounts.
- **Feed Page**: View posts from followed users.
- **Dark Mode**: Toggle between light and dark themes.
- **Responsive Design**: Ensures compatibility across various devices.
- **Secure Data Management**: Secure handling and storage of user data.
- **Search Feature**: Search for users and posts within the platform.

### Work in Progress
- **Real-time Chat Feature**: Allow users to communicate in real-time.
- **Notification System**: Real-time notifications for user interactions.
- **Deployment**: Preparing the application for deployment to a cloud platform.

## Technologies Used
- **Frontend**: React, Redux
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JSON Web Tokens (JWT)
- **Development Tools**: Visual Studio Code, Postman
- **Version Control**: Git, GitHub

## Installation

### Prerequisites
- Node.js
- npm or yarn
- MongoDB

### Clone the Repository
```bash
git clone https://github.com/Ravij-p/Talkies.git
cd Talkies
```

### Install Dependencies
```bash
# For backend
cd server
npm install

# For frontend
cd ../talkies
npm install

```

### Run the Application
```bash
# Run frontend server
cd talkies
npm start

# Run backend development server
cd ../server
npm run dev
```
Visit `http://localhost:3000` to see the application in action.

## Usage
1. **Sign Up**: Create a new account by providing a username and password.
2. **Login**: Access your account using your credentials.
3. **Create Post**: Share your thoughts by creating new posts.
4. **Interact**: Like, comment, and save posts from other users.
5. **Profile Management**: Update your profile information.
6. **Privacy Settings**: Set your account to public or private.
7. **Follow Users**: Follow other users to see their posts in your feed.
8. **Manage Follow Requests**: Accept or reject follow requests for private accounts.
9. **Feed Page**: View posts from users you follow.
10. **Dark Mode**: Switch between light and dark themes for better user experience.
11. **Search**: Search for users and posts within the platform.
